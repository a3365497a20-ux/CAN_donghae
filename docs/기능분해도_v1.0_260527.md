## 1. 기능 분해 개요

시스템 기능을 **원자 단위(Atomic)**로 계층적으로 분해한다. 최상위 시스템 기능을 대분류로 나누고, 각 대분류를 중분류 → 원자 기능 단위까지 세분화한다.

---

## 2. 기능 분해도 (계층 구조)

```
ICT CampusNav
│
├── F1. 사용자 인증
│     ├── F1-1. 로그인
│     │         ├── F1-1-1. 아이디/비밀번호 입력 검증
│     │         ├── F1-1-2. DB 사용자 조회
│     │         ├── F1-1-3. 역할 일치 확인
│     │         ├── F1-1-4. 세션 생성 (loginUser / loginName / loginRole)
│     │         └── F1-1-5. 역할별 메인 페이지 리다이렉트
│     ├── F1-2. 로그아웃
│     │         ├── F1-2-1. 세션 무효화
│     │         └── F1-2-2. 로그인 페이지 이동
│     ├── F1-3. 회원가입
│     │         ├── F1-3-1. 아이디 중복 체크
│     │         ├── F1-3-2. 비밀번호 일치 확인
│     │         └── F1-3-3. users 테이블 INSERT
│     ├── F1-4. 게스트 진입
│     │         └── F1-4-1. 게스트 세션 설정 → main_guest.jsp 이동
│     └── F1-5. 외부 방문자 진입
│               └── F1-5-1. visitor 세션 설정 → main_visitor.jsp 이동
│
├── F2. 자산 검색 및 조회
│     ├── F2-1. 자산 통합 검색
│     │         ├── F2-1-1. 키워드 검색 (품목명 / 자산번호 / 위치 LIKE)
│     │         ├── F2-1-2. 분류 필터 (공기구비품 / 집기비품 / 무형고정자산)
│     │         ├── F2-1-3. 상태 필터 (사용 / 사용중 / 폐기)
│     │         └── F2-1-4. 페이지네이션 (20건씩)
│     └── F2-2. 자산 상세 조회
│               ├── F2-2-1. 자산 기본 정보 표시 (전체 컬럼)
│               ├── F2-2-2. 이관 이력 타임라인 표시
│               ├── F2-2-3. 향후 예약 현황 표시
│               └── F2-2-4. 지도 위치 표시 (GPS 연동 후 활성화)
│
├── F3. 자산 예약
│     ├── F3-1. 예약 신청
│     │         ├── F3-1-1. 날짜·시간 선택
│     │         ├── F3-1-2. AJAX 중복 체크 (빨강/초록 피드백)
│     │         └── F3-1-3. reservations 테이블 INSERT
│     └── F3-2. 기존 예약 현황 조회
│               └── F3-2-1. 해당 자산 예약 목록 표시
│
├── F4. 강의실 예약
│     ├── F4-1. 강의실 목록 조회
│     │         └── F4-1-1. rooms 테이블 조회 (건물 / 유형 / 수용인원)
│     ├── F4-2. 강의실 예약 신청
│     │         ├── F4-2-1. 날짜·시간·목적·연락처 입력
│     │         ├── F4-2-2. AJAX 중복 체크
│     │         └── F4-2-3. room_reservations 테이블 INSERT
│     └── F4-3. 예약 상태 관리 (관리자)
│               └── F4-3-1. 상태 변경 (예약완료 / 이용중 / 완료 / 취소)
│
├── F5. 이관 이력 조회
│     ├── F5-1. 이관 목록 조회
│     │         └── F5-1-1. asset_transfer 최근 50건 조회 (출발→도착 화살표)
│     └── F5-2. 이관 상세 패널
│               └── F5-2-1. 행 클릭 시 출발/도착 부서 흐름 패널 표시
│
├── F6. 교수 자원 관리
│     ├── F6-1. 교수 목록 조회
│     │         └── F6-1-1. professors + 과목 + 주특기 JOIN 카드 표시
│     ├── F6-2. 과목 등록
│     │         └── F6-2-1. prof_subjects INSERT (과목명/코드/학년/학기/강의실/시간)
│     └── F6-3. 주특기·연구분야 관리
│               ├── F6-3-1. 태그 추가 (prof_skills INSERT)
│               └── F6-3-2. 태그 삭제 (prof_skills DELETE)
│
├── F7. 관리자 기능
│     ├── F7-1. 통계 대시보드
│     │         └── F7-1-1. 자산/이관/폐기/예약 건수 실시간 SELECT COUNT
│     ├── F7-2. 자산 관리
│     │         ├── F7-2-1. 자산 신규 등록 (assets INSERT)
│     │         └── F7-2-2. 자산 정보 수정 (assets UPDATE)
│     ├── F7-3. 폐기 관리
│     │         └── F7-3-1. 폐기 자산 목록 조회
│     └── F7-4. 예약 관리
│               ├── F7-4-1. 전체 예약 목록 조회 (유형별 필터)
│               └── F7-4-2. 예약 취소 처리
│
└── F8. 캠퍼스 길찾기
      ├── F8-1. 외부 경로 안내
      │         ├── F8-1-1. 출입구 마커 표시 (Canvas 애니메이션)
      │         ├── F8-1-2. 출발지/목적지 선택
      │         ├── F8-1-3. 경로 데이터 AJAX 조회 (getRoutes.jsp)
      │         └── F8-1-4. Canvas polyline 경로 렌더링
      ├── F8-2. 층별 실내 경로 안내
      │         ├── F8-2-1. 층 평면도 표시
      │         └── F8-2-2. floor_routes 경로 렌더링
      └── F8-3. 관리자 경로 등록
                ├── F8-3-1. 출입구 핀 위치 설정 (saveEntrancePin.jsp)
                ├── F8-3-2. 평면도 클릭 경유점 등록 (saveRoute.jsp)
                └── F8-3-3. 층별 실내 경로 등록 (saveFloorRoute.jsp)
```
