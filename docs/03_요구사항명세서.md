# 요구사항 명세서

**프로젝트명:** ICT 교내 자원 내비게이션 시스템 (CampusNav)
**기관:** ICT폴리텍대학 AI소프트웨어학과
**팀원:** 박승순 · 권동해 · 원태연 · 이수혁
**작성일:** 2026-05-27
**버전:** v1.0

---

## 1. 문서 목적

본 문서는 ICT CampusNav 시스템 개발에 필요한 기능 요구사항(FR)과 비기능 요구사항(NFR)을 정의하고, 제약사항 및 사용자 스토리를 명세한다.

---

## 2. 요구사항 우선순위 기준

| 우선순위 | 기준 |
|----------|------|
| 상 (High) | 시스템 핵심 기능, 미구현 시 서비스 불가 |
| 중 (Medium) | 주요 부가 기능, 사용성에 직접 영향 |
| 하 (Low) | 선택적 기능, 향후 확장 대상 |

---

## 3. 기능 요구사항 (Functional Requirements)

### FR-01 사용자 인증 및 권한 관리

| 요구사항 ID | 요구사항명 | 상세 내용 | 역할 | 우선순위 |
|-------------|-----------|-----------|------|----------|
| FR-01-01 | 로그인 | 사용자 ID·비밀번호 입력, 역할 버튼 선택 후 DB 조회 인증 | 전체 | 상 |
| FR-01-02 | 역할 기반 페이지 라우팅 | 로그인 성공 시 역할(student/assistant/professor/admin/visitor)에 따라 해당 메인 페이지로 자동 이동 | 전체 | 상 |
| FR-01-03 | 역할-계정 불일치 오류 처리 | 선택 역할과 DB 역할이 다를 경우 오류 메시지 표시 후 재입력 유도 | 전체 | 상 |
| FR-01-04 | 로그아웃 | 세션 무효화 후 로그인 페이지로 이동 | 전체 | 상 |
| FR-01-05 | 회원가입 | 아이디 중복 체크, 비밀번호 확인 후 DB에 계정 저장 | 미가입 사용자 | 상 |
| FR-01-06 | 아이디 저장 | 쿠키 기반 아이디 30일 저장 | 전체 | 하 |
| FR-01-07 | 게스트 접속 | 로그인 없이 자산 검색·상세 조회 가능 (예약·길찾기 기능 잠금) | 게스트 | 중 |
| FR-01-08 | 외부 방문자 접속 | 계정 없이 공간 예약 및 캠퍼스 길찾기만 이용 가능 | 외부 방문자 | 중 |

### FR-02 자산 검색 및 조회

| 요구사항 ID | 요구사항명 | 상세 내용 | 역할 | 우선순위 |

| FR-02-01 | 자산 검색 | 키워드(자산번호·품목명·위치), 분류, 상태를 복합 조건으로 검색 (전체 8,401건 대상) | 전체 | 상 |

| FR-02-05 | 향후 예약 현황 표시 | 자산 상세 페이지에서 해당 자산의 예약 목록 표시 | 전체 | 중 |
| FR-02-06 | 자산 위치 지도 표시 | GPS 좌표 연동 후 자산 위치를 지도 핀으로 시각화 | 전체 | 하 |

| 요구사항 ID | 내용 | 기준 |
|-------------|------|------|

## 5. 제약사항

| 항목 | 내용 |
|------|------|
| 기술 스택 고정 | JSP/Servlet + MySQL 8.x + Apache Tomcat 9.x 환경 고정 |
| 외부 API 미사용 | Google Maps API는 키 발급 후 연동 예정, 현재 미적용 |
| 수동 컴파일 필요 | LoginServlet 등 서블릿은 javac 수동 컴파일 및 compile.bat 실행 필요 |
| 비밀번호 정책 | 현재 평문 저장 방식 → 추후 해싱 처리 필요 |
| 실내 GPS 미적용 | 하드웨어 설치 전까지 실내 위치 기능 비활성 |
| 단일 서버 | 현재 로컬 환경(localhost) 기반 운영, 외부 접속 불가 |

## 6. 사용자 스토리 (User Stories)

| 역할 | 스토리 |
|------|--------|
| 학부생 | 자산번호나 품목명으로 교내 자산을 검색하고, 사용 가능한 자산을 예약하고 싶다. |
| 조교 | 자산 이관 이력을 조회하고, 강의실 예약 현황을 확인하고 싶다. |
| 교수 | 내 담당 과목과 연구분야를 등록·수정하고, 필요한 자산을 예약하고 싶다. |
| 운영관리자 | 전체 자산·예약·이관·폐기 통계를 한눈에 보고, 자산 상태를 관리하고 싶다. |
| 게스트 | 로그인 없이 교내 자산이 어디에 있는지 검색하고 상세 정보를 확인하고 싶다. |
| 외부 방문자 | 계정 없이 강의실을 예약하고 캠퍼스 길을 찾고 싶다. |
