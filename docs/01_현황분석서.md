## 2. 자산 현황

### 2.1 유형별 자산 현황

교내 보유 자산은 총 **8,401건**이며, 아래 세 가지 유형으로 분류된다.

| 자산 유형 | 건수 | 비고 |
|-----------|------|------|
| 공기구비품 | 3,368건 | 기자재, 장비, 전산기기 등 |
| 집기비품 | 4,737건 | 가구, 사무용품, 생활비품 등 |
| 무형고정자산(SW) | 296건 | 소프트웨어 라이선스, OS 등 |
| **합계** | **8,401건** | |

### 2.2 변동 이력 현황

| 이력 유형 | 건수 | 비고 |
|-----------|------|------|
| 이관 이력 | 140건 | 부서 간 자산 이동 기록 |
| 폐기 이력 | 2,721건 | 폐기 처리된 자산 기록 |

### 2.3 자산 속성 현황

각 자산은 아래 속성으로 관리된다.

| 속성 | 설명 | 예시 |
|------|------|------|
| 자산번호 | 고유 식별 코드 | `2511C0043` |
| 자산 유형 | 공기구비품 / 집기비품 / 무형고정자산 | 공기구비품 |
| 품목명 | 자산 명칭 | 노트북 |
| 제조사 / 모델명 | 제조사 및 모델 | NT960XHA-KG73G |
| 취득일 | 구매 일자 | 2025-11-03 |
| 취득가액 / 현재가액 | 취득 당시 가액 및 현재 가치 | 2,130,000 / 2,094,500 |
| 내용연수 | 감가상각 기간 | 5년 |
| 위치 경로 | 건물·층·호실 파이프 구분 경로 | `ICT폴리텍대학\|대학본부\|4층\|` |
| 호실 | 세부 호실 정보 | 401호 |
| 담당 학과 | 자산 관리 학과 | IT융합학과 |
| 담당자 | 담당 교직원 | 박진용 |
| 상태 | 자산 사용 상태 | 사용 / 사용중 / 폐기 |

### 2.4 자산 위치 관리 현황

- 위치 정보는 `ICT폴리텍대학|제1공학관|1101호|` 형태의 **텍스트 경로**로만 저장
- 실내 GPS 좌표(위도·경도)는 현재 **미연동 상태** (향후 제1·2공학관 GPS 설치 예정)
- 지도를 통한 시각적 위치 파악이 불가능하여 자산 물리적 탐색이 어려움

## 3. 공간(강의실) 현황

### 3.1 공간 유형

| 유형 | 수용 인원 | 특징 |
|------|-----------|------|
| 강의실 | 40 ~ 45명 | 일반 강의용 |
| 컴퓨터실 | 30 ~ 32대 | PC 실습용, AI/네트워크 특화 포함 |
| 세미나실 | 소규모 | 회의·세미나용 |

### 3.2 건물별 공간 분포

| 건물 | 주요 공간 |
|------|-----------|
| 제1공학관 | 강의실, 컴퓨터실, 세미나실, 북카페 |
| 제2공학관 | 강의실, 컴퓨터실, 정보처리실, AI/네트워크 실습실 |
| 대학본부 | 교수 연구실, 행정 사무실, 방송실 |
| 학생회관 | 학생 편의시설 |
| 산학협력관 | 산업기술연구소, 기숙사형 공간 |

예약 가능한 전체 공간은 **64개**이며, 제1·2공학관에 집중되어 있다.

## 5. 현행 업무 프로세스

### 5.1 자산 관리 프로세스

```
자산 구매/등록 → DB에 자산 정보 입력 → 부서 배치(위치 텍스트 경로 입력)
      ↓
  자산 이관 시 → 이관 이력 DB 기록 (출발부서 → 도착부서)
      ↓
  자산 폐기 시 → 폐기 이력 DB 기록
```

### 5.2 강의실 예약 프로세스

```
사용자 로그인 → 강의실 목록 조회 → 날짜·시간 선택
      ↓
  실시간 중복 체크 → 가능: 초록 표시 / 중복: 빨간 경고
      ↓
  예약 정보 입력(목적, 연락처) → DB 저장 → 예약완료
```

## 6. 현행 시스템 문제점

| 문제 유형 | 내용 | 영향 |
|-----------|------|------|
| 위치 가시성 부재 | 자산 위치가 텍스트 경로로만 저장 | 실내 탐색 어려움 |
| 통합 관리 부재 | 자산 검색·예약·이관 조회가 통합된 시스템 없이 수작업 관리 | 업무 비효율 |
| 권한 구분 미흡 | 역할별 기능 접근 제어가 체계적으로 되어 있지 않음 | 정보 보안 위험 |
| 예약 중복 관리 | 자산·강의실 예약 시 실시간 중복 확인 불가 | 예약 충돌 발생 |
| 보안 취약 | 비밀번호 평문 저장 | 계정 정보 유출 위험 |
| 지도 미연동 | Google Maps 및 실내 GPS 미적용 | 길찾기 기능 미완성 |
