## 4. 의존성 관리 계획 (Dependency Management)

### 4.1 모듈 간 의존성

```
DB 구축
  └─► 공통 유틸 (DBUtil.java)
        └─► 인증 모듈
              └─► 역할별 메인
                    ├─► 자산 검색 → 자산 예약
                    ├─► 강의실 예약
                    ├─► 이관 조회
                    ├─► 교수 자원
                    ├─► 관리자 기능
                    └─► 길찾기
```

### 4.2 외부 의존성

| 의존 항목 | 버전 | 획득 방법 | 필수 여부 |
|-----------|------|-----------|-----------|
| Apache Tomcat | 9.0.x | 공식 사이트 다운로드 | 필수 |
| MySQL | 8.x | 공식 사이트 설치 | 필수 |
| mysql-connector-j | 최신 | Maven Repository | 필수 |
| Bootstrap | 5.3.3 | CDN | 필수 |
| Bootstrap Icons | 최신 | CDN | 필수 |
| Google Maps API | — | API 키 발급 후 | 선택 (향후) |
