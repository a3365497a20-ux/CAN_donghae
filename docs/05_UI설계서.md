# UI 설계서

**프로젝트명:** ICT 교내 자원 내비게이션 시스템 (CampusNav)
**기관:** ICT폴리텍대학 AI소프트웨어학과
**팀원:** 박승순 · 권동해 · 원태연 · 이수혁
**작성일:** 2026-05-27
**버전:** v1.0

---

## 1. UI 설계 개요

### 1.1 디자인 시스템

| 항목 | 내용 |
|------|------|
| 디자인 스타일 | ppd4 — 화이트 베이스 + 블루 포인트 |
| 기본 폰트 크기 | 15px |
| UI 프레임워크 | Bootstrap 5.3.3 |
| 아이콘 | Bootstrap Icons |
| 폰트 | DM Sans (본문), DM Mono (코드·배지), Noto Sans KR (한글) |
| 반응형 | Bootstrap Grid 기반, 모바일·PC 대응 |

### 1.2 공통 색상 변수

| 변수명 | 색상 | 용도 |
|--------|------|------|
| `--blue` | #1d4ed8 | 주요 포인트, 버튼, 링크 |
| `--blue-lt` | #eff6ff | 파란 배경 강조, 검색바 shadow |
| `--blue-md` | #bfdbfe | 테이블 헤더, 검색바 테두리 |
| `--teal` | #0d9488 | 길찾기·네비게이션 포인트 |
| `--green` | #16a34a | 예약 가능 상태 (초록) |
| `--red` | #dc2626 | 예약 중복 경고 (빨강) |
| `--amber` | #d97706 | 외부 방문자 안내, 경고 |
| `--txt` | #0f172a | 기본 텍스트 |
| `--txt2` | #475569 | 보조 텍스트 |
| `--txt3` | #94a3b8 | 희미한 텍스트, 플레이스홀더 |
| `--line` | #e2e8f0 | 구분선 |
| `--white` | #ffffff | 카드·배경 |

### 1.3 공통 레이아웃 구조

```
┌─────────────────────────────────────────────┐
│              상단 내비게이션 (topnav)         │
│  [로고: ICT CAN]        [역할칩] [홈] [로그아웃] │
├─────────────────────────────────────────────┤
│                  Hero 섹션                   │
│  eyebrow 텍스트 / 제목 / 설명 / 검색바 or 버튼  │
├─────────────────────────────────────────────┤
│                  본문 (shell)                │
│         카드(card) 기반 콘텐츠 영역            │
├─────────────────────────────────────────────┤
│                  푸터 (footer)               │
│  [로고]  [팀원명]  [저작권]                   │
└─────────────────────────────────────────────┘
```

## 3. 화면별 상세 설계

### UI-01 로그인 화면

**파일:** `campuslogin.jsp`

**레이아웃:**
```
┌──────────────────────────────┐
│  [ICT CAN 로고]               │
│  역할 선택 버튼               │
│  [학부생] [조교] [교수] [관리자] [외부인] │
├──────────────────────────────┤
│  아이디 입력 (아이콘 + 텍스트필드) │
│  비밀번호 입력 (아이콘 + 눈 토글) │
│  [아이디 저장 체크박스]          │
│  [로그인 버튼]                  │
│  ───── 또는 ─────              │
│  [회원가입 버튼]                │
│  [로그인 없이 둘러보기(게스트)]    │
│  테스트 계정 힌트 박스            │
└──────────────────────────────┘
```

**주요 UI 요소:**

| 요소 | 유형 | 설명 |
|------|------|------|
| 역할 선택 버튼 | radio-style button | 클릭 시 해당 테스트 계정 자동 입력 |
| 아이디 필드 | text input + 아이콘 | is-error 클래스로 오류 시 빨간 테두리 |
| 비밀번호 필드 | password input + 눈 토글 | 비밀번호 표시/숨김 전환 |
| 오류 메시지 | inline alert | bi-exclamation-circle-fill 아이콘 + 빨간 텍스트 |
| 외부인 패널 | 별도 패널 전환 | 외부인 버튼 클릭 시 staff 패널 → visitor 패널로 전환 |

### UI-09 자산 검색 화면

**파일:** `search.jsp`

**레이아웃:**
```
┌────────────────────────────────────────┐
│  검색바 (keyword + 분류 select + 상태 select + 검색 버튼) │
├────────────────────────────────────────┤
│  검색 결과 테이블                         │
│  [자산번호] [품목명] [분류] [위치] [상태]  │
│  행 클릭 → detail.jsp 이동               │
├────────────────────────────────────────┤
│  페이지네이션 (20건씩)                    │
│  [◀ 이전] [1][2][3]...[다음 ▶]          │
└────────────────────────────────────────┘
```

**검색 필터:**

| 필터 | 유형 | 선택지 |
|------|------|--------|
| 키워드 | text input | 자산번호, 품목명, 위치 검색 |
| 분류 | select | 전체 / 공기구비품 / 집기비품 / 무형고정자산 |
| 상태 | select | 전체 / 사용 / 사용중 / 폐기 |

**테이블 상태 배지:**

| 상태 | 배지 색상 |
|------|-----------|
| 사용 | badge-ok (초록) |
| 사용중 | badge-busy (파랑) |
| 폐기 | badge-warn (빨강) |

## 4. 화면 전환 흐름

```
campuslogin.jsp (로그인)
    │
    ├─ student   → main_student.jsp
    │                  └─ search.jsp → detail.jsp → reserve.jsp
    │
    ├─ assistant → main_assistant.jsp
    │                  └─ search.jsp / transfer.jsp / room_reserve.jsp
    │
    ├─ professor → main_professor.jsp
    │                  └─ professor.jsp / search.jsp
    │
    ├─ admin     → main_admin.jsp
    │                  └─ asset_manage.jsp / disposal_admin.jsp
    │                     reservations_admin.jsp / search.jsp
    │
    ├─ guest     → main_guest.jsp
    │                  └─ search.jsp → detail.jsp (예약 불가)
    │
    └─ visitor   → main_visitor.jsp
                       └─ room_reserve.jsp / navigationTest1.jsp
```
